# Use Cases

AnyKeyboard is built for real-world multilingual typing.

It supports:
- 100+ keyboard layouts
- Speed writing utility (all languages)
- Phonetic typing for major languages

---

## Language Learning

Practice writing in a new language without changing your system keyboard.
Phonetic typing (when available) can help beginners type before they learn
the native layout.

---

## Writing & Translation

Type accents, scripts, and foreign characters quickly when writing articles,
emails, or translated content. The speed writing utility helps reduce friction
during longer writing sessions.

---

## SEO & Localization

Useful for:
- Writing localized content
- Checking character accuracy
- Working across multiple languages

---

## Travel & Communication

Type messages in local languages when traveling or communicating internationally.

---

## Shared or Restricted Devices

Ideal for:
- School computers
- Public computers
- Work devices with locked settings
